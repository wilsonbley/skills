# Skills

Custom global skills for AI coding agents.

## Install

```bash
git clone <repo-url> ~/Developer/skills
cd ~/Developer/skills
./install.sh
```

## Adding a new skill

1. Create a new directory with a `SKILL.md` file at the root of the repo
2. Run `./install.sh`
